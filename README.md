# Spotify UI Clone

A responsive Spotify interface clone built with HTML, CSS and JavaScript.

This project was developed for learning purposes, focusing on layout, responsiveness and JavaScript interactions by recreating a Spotify-like interface.

---

## Features

* Spotify-inspired layout
* Responsive design (desktop and mobile)
* Sidebar navigation
* Music player UI
* Interactive elements using JavaScript
* Dynamic UI behavior

---

## Tech Stack

* HTML
* CSS (modular structure)
* JavaScript (vanilla)

---

## What I Practiced

* Responsive design
* Layout replication from a real-world application
* Flexbox and CSS Grid
* DOM manipulation
* Event handling with JavaScript
* Building interactive UI components

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/elyabebat0129/spotify-ui-clone.git
```

2. Open the project folder

3. Open `index.html` in your browser
